Tanzu Technical Field Update 雲原生相關重點技術更新

Created2021-04-19|Updated2022-02-22|work hack那根樹枝 Tanzu
|Post View:

/ 上週重點技術更新 /

- Kubernetes 1.21

Kubernetes 最新版本於 4 月 8 日發佈了! 詳細全文: https://kubernetes.io/blog/2021/04/08/kubernetes-1-21-release-announcement/

- TMC (Tanzu Mission Control) now enable TKG installation

現在使用 TMC 安裝 TKG on vSphere 功能已經正式開放給全部用戶使用! (限vSphere 7.0 & 6.7)
用戶可以直接登入您的TMC環境來執行。

/ 好文分享 /

你知道擁有越高的 CPU 超額乘載 (overcommitment) 能夠讓TKG Cluster 擁有更佳的總體效能嗎？

今年 3 月，VMware 效能分析團隊 (performance team) 出了一份技術報告，提供系統管理員與開發人員最佳化工作負載效能的建議，包含：

  • Sizing VMware TKG clusters and selecting virtual machine classes for worker nodes
  • Configuring pod and container resource specifications
  • Choosing the right level of CPU and memory over-commitment with the least impact on workload performance

報告裡面指出，超額承載明顯提升了資源利用率。當然，還是要記得去監控 vSphere key metrics 來避免爭奪資源導致效能反而下降。詳細可以查看技術報告全文：https://www.vmware.com/content/dam/digitalmarketing/vmware/en/pdf/techpaper/performance/vsphere-tanzu-kubernetes-perf.pdf

或是閱讀Harold Rosenberg 撰寫的精華版文章：
https://blogs.vmware.com/performance/2021/04/performance-best-practices-for-kubernetes-with-vmware-tanzu.html?src=so_602d54e7e837c&cid=7012H000001KW9I
